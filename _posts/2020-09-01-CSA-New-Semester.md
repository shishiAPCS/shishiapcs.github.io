## 在线错题本 Yuque / 语雀

访问 [yuque.com](https://www.yuque.com/)

### 第一步：注册 / 登录账号

* 如果已有账号，点击 `登录`。
* 如果没有账号，先注册一个语雀账号。

### 第二步：新建知识库

登录后，进入语雀首页或工作台。

常见入口：

* 点击页面上的 `+` / `新建`
* 选择 `新建知识库`

### 第三步：设置知识库名称

知识库名称请使用下面的格式：

```
APCSA-Errorlog-英文名
```

例如：

```
APCSA-Errorlog-Sky
```

注意：

* `APCSA` 大写。
* `Errorlog` 表示错题本 / 错误记录。
* 最后改成自己的英文名。
* 大小写必须和参考一致

### 第四步：完成创建

* 归属：选择个人账号即可。
* 点击 `创建` / `新建` 完成。

### 第五步：添加 Mr.Mou 为协作者

创建知识库后，需要把 Mr.Mou 添加为协作者，并设置为**可编辑**权限。

操作步骤：

* 进入你刚刚创建的知识库。
* 在知识库页面右上角，找到 `分享` / `协作` / `成员管理` 相关入口。
* 选择 `添加协作者`。
* 使用搜索添加，搜索：

```
mr.mou
```

* 找到 Mr.Mou 后，添加为协作者。
* 权限请选择：

```
可编辑
```

注意：

* 不要只设置为 `仅阅读`。
* 添加完成后，老师应该可以进入你的知识库，查看并编辑你的错题本。

### 第六步：新建第一篇文档

进入知识库后，新建第一篇文档，标题可以写：

```
Unit 1 MCQ
```
