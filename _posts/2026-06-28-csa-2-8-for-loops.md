A `while` loop separates initialization, testing, and updating into different places. A `for` loop places those three parts together in one header, making counter-controlled repetition easier to read.

`while` 循环把初始化、条件检查和变量更新写在不同位置，而 `for` 循环会把这三个部分集中写在循环头中，更适合控制明确次数的重复。

> **A `for` loop initializes once, checks the condition before every iteration, and updates after every completed loop body.**
> **`for` 循环只初始化一次；每次迭代前检查条件；每次循环体执行完后更新变量。**

# AP CSA 2.8 for 循环<br>For Loops

## 循环头结构<br>Loop Header

* `for (初始化; 条件; 更新)`<br>`for (initialization; condition; update)`
* 三个部分由两个分号分隔<br>Two semicolons separate the three parts
* 循环变量通常作为计数器<br>The loop variable usually acts as a counter

## 执行顺序<br>Execution Order

* 初始化只执行一次<br>Initialization runs once
* 每次迭代前检查条件<br>The condition is checked before each iteration
* 循环体执行后再更新变量<br>The update runs after the loop body
* 条件为假时退出循环<br>The loop stops when the condition is false

## 常见计数模式<br>Common Counting Patterns

* `i = 0; i < n; i++`：执行 `n` 次<br>Runs `n` times
* `i = 1; i <= n; i++`：执行 `n` 次<br>Runs `n` times
* `i += 2`：每次增加 2<br>Counts by twos
* `i--`：反向计数<br>Counts backward

## 与 while 等价<br>Equivalent while Loop

* `for` 和 `while` 可以表达相同逻辑<br>Both loops can express the same logic
* 初始化写在 `while` 前<br>Initialization appears before the `while`
* 更新写在 `while` 循环体末尾<br>Update appears at the end of the body

## 循环边界<br>Loop Boundaries

* `<` 不包含结束值<br>`<` excludes the ending value
* `<=` 包含结束值<br>`<=` includes the ending value
* 起点、终点和更新共同决定次数<br>Start, boundary, and update determine the iteration count

## 常见错误<br>Common Errors

* 更新方向与条件不匹配<br>The update moves in the wrong direction
* 使用错误的 `<` 或 `<=`<br>Using the wrong boundary operator
* 忘记更新在循环体之后执行<br>Forgetting that the update runs after the body
* 循环头中的分号位置错误<br>Misplacing semicolons in the header

## AP 考试重点<br>AP Exam Focus

* 追踪循环变量和输出<br>Trace loop variables and output
* 判断准确的迭代次数<br>Determine the exact number of iterations
* 比较不同循环头的结果<br>Compare the effects of different headers
* 判断 `for` 与 `while` 是否等价<br>Identify equivalent `for` and `while` loops

# 1. Trace the Values Generated by the Header<br>追踪循环头产生的数值

What is printed?

下面的代码会输出什么？

```java
int result = 1;

for (int i = 3; i < 6; i += 2)
{
    result *= i;
}

System.out.println(result);
```

**Answer:**

```text
15
```

The loop variable takes these values:

循环变量会依次取下面的值：

```text
i = 3
i = 5
```

Trace the loop:

逐次追踪：

| Stage                          | `i` | Condition                | `result` |
| ------------------------------ | --: | ------------------------ | -------: |
| Start<br>开始                    | `3` | `3 < 6` → `true`         |      `1` |
| After iteration 1<br>第 1 次迭代后  | `5` | Update completed<br>完成更新 |      `3` |
| Before iteration 2<br>第 2 次迭代前 | `5` | `5 < 6` → `true`         |      `3` |
| After iteration 2<br>第 2 次迭代后  | `7` | Update completed<br>完成更新 |     `15` |
| Final check<br>最后检查            | `7` | `7 < 6` → `false`        |     `15` |

The update `i += 2` runs after the loop body. Therefore, the body uses `i = 3` and `i = 5`.

更新语句 `i += 2` 会在循环体执行后运行。因此，循环体中使用的值是 `3` 和 `5`。

# 2. The Three Parts of a `for` Loop<br>`for` 循环的三个部分

The general form is:

基本结构如下：

```java
for (initialization; booleanExpression; update)
{
    statements;
}
```

Example:

```java
for (int count = 1; count <= 5; count++)
{
    System.out.println(count);
}
```

```text
1
2
3
4
5
```

The header contains three parts:

循环头包含三个部分：

| Part                        | Example         | Purpose                                                                   |
| --------------------------- | --------------- | ------------------------------------------------------------------------- |
| Initialization<br>初始化       | `int count = 1` | Creates the loop variable and gives it a starting value.<br>创建循环变量并设置初始值。 |
| Boolean expression<br>布尔表达式 | `count <= 5`    | Decides whether the next iteration runs.<br>决定是否执行下一次迭代。                  |
| Update<br>更新                | `count++`       | Changes the loop variable after each iteration.<br>每次迭代后改变循环变量。           |

The two semicolons in the header are required:

循环头中的两个分号是必需的：

```java
for (initialization; condition; update)
```

Do not replace them with commas.

不能使用逗号代替这两个分号。

# 3. Execution Order of a `for` Loop<br>`for` 循环的执行顺序

Consider:

观察下面的代码：

```java
for (int i = 2; i <= 6; i += 2)
{
    System.out.print(i + " ");
}
```

```text
2 4 6
```

Java follows this order:

Java 会按照下面的顺序执行：

1. Run the initialization once.
   初始化只执行一次。

   ```java
   int i = 2;
   ```

2. Check the condition.
   检查条件。

   ```java
   i <= 6
   ```

3. If the condition is true, run the entire loop body.
   如果条件为真，执行完整的循环体。

4. Run the update.
   执行更新语句。

   ```java
   i += 2;
   ```

5. Return to the condition.
   返回循环条件，再次检查。

6. Stop when the condition becomes false.
   当条件变为假时，退出循环。

The important order is:

最重要的执行顺序是：

```text
initialize → test → body → update → test → body → update
```

The update does **not** run before the first iteration.

第一次迭代前不会执行更新语句。

# 4. Common Counting Patterns<br>常见计数模式

## Count from `0` up to, but not including, `n`<br>从 `0` 数到 `n - 1`

```java
for (int i = 0; i < n; i++)
{
    statements;
}
```

When `n` is positive, this loop runs `n` times.

当 `n` 为正数时，这个循环会执行 `n` 次。

For `n = 5`, the values are:

当 `n = 5` 时：

```text
0 1 2 3 4
```

## Count from `1` through `n`<br>从 `1` 数到 `n`

```java
for (int i = 1; i <= n; i++)
{
    statements;
}
```

For `n = 5`, the values are:

当 `n = 5` 时：

```text
1 2 3 4 5
```

This loop also runs five times.

这个循环同样执行五次。

| Header pattern           | Values when `n = 5` | Iterations |
| ------------------------ | ------------------- | ---------: |
| `int i = 0; i < n; i++`  | `0, 1, 2, 3, 4`     |        `5` |
| `int i = 1; i <= n; i++` | `1, 2, 3, 4, 5`     |        `5` |

Do not decide the number of iterations by looking only at the ending value. Check the starting value, condition, and update together.

不能只看结束值来判断循环次数。必须同时检查初始值、条件和更新方式。

# 5. Counting Backward and Skipping Values<br>反向计数与跳跃计数

The update does not have to be `i++`.

更新语句不一定是 `i++`。

## Counting backward<br>反向计数

```java
for (int i = 5; i > 0; i--)
{
    System.out.print(i + " ");
}
```

```text
5 4 3 2 1
```

All three parts support the backward direction:

三个部分都与反向计数保持一致：

* Start at `5`.

* Continue while `i > 0`.

* Decrease `i`.

* 从 `5` 开始。

* 当 `i > 0` 时继续。

* 每次减小 `i`。

## Counting by twos<br>每次增加 2

```java
for (int i = 0; i <= 10; i += 2)
{
    System.out.print(i + " ");
}
```

```text
0 2 4 6 8 10
```

## Counting backward by twos<br>每次减少 2

```java
for (int i = 5; i >= 1; i -= 2)
{
    System.out.print(i + " ");
}
```

```text
5 3 1
```

When counting backward, the condition and update must both point toward smaller values.

反向计数时，循环条件和更新方式都必须与数值减小的方向一致。

# 6. Equivalent `for` and `while` Loops<br>等价的 `for` 与 `while` 循环

A `for` loop can be rewritten as a `while` loop.

`for` 循环可以改写成等价的 `while` 循环。

## `for` version<br>`for` 版本

```java
for (int i = 3; i > 0; i--)
{
    System.out.println(i);
}
```

## Equivalent `while` version<br>等价的 `while` 版本

```java
int i = 3;

while (i > 0)
{
    System.out.println(i);
    i--;
}
```

Both versions print:

两个版本都会输出：

```text
3
2
1
```

The parts map directly:

各部分的对应关系如下：

| `for` loop part             | Location in the `while` loop             |
| --------------------------- | ---------------------------------------- |
| Initialization<br>初始化       | Before the `while` loop<br>写在 `while` 前面 |
| Boolean expression<br>布尔表达式 | In the `while` header<br>写在 `while` 循环头中 |
| Update<br>更新                | At the end of the loop body<br>写在循环体末尾   |

When comparing two loops on an AP question, check whether they generate the same loop-variable values in the same order.

在 AP 题目中比较两个循环时，应检查它们是否按照相同顺序产生相同的循环变量值。

# 7. Boundary Values and Iteration Counts<br>边界值与迭代次数

How many times does this loop print `"*"`?

下面的循环会输出多少个 `"*"`？

```java
for (int i = 3; i <= 9; i++)
{
    System.out.print("*");
}
```

**Answer: Seven times.**

**答案：七次。**

The values are:

循环变量的值为：

```text
3 4 5 6 7 8 9
```

Count the actual values rather than subtracting carelessly.

应该数出循环变量实际取到的值，而不是直接进行不完整的减法。

For a loop that increases by `1` and includes both endpoints:

对于每次增加 `1`，并且包含起点与终点的循环：

```text
number of iterations = ending value - starting value + 1
```

Here:

这里：

```text
9 - 3 + 1 = 7
```

Compare:

比较：

| Loop header              | Values                | Iterations |
| ------------------------ | --------------------- | ---------: |
| `int i = 3; i < 9; i++`  | `3, 4, 5, 6, 7, 8`    |        `6` |
| `int i = 3; i <= 9; i++` | `3, 4, 5, 6, 7, 8, 9` |        `7` |

Changing `<` to `<=` often adds one iteration.

把 `<` 改成 `<=`，通常会增加一次迭代。

比较下面两个循环。

做 AP 题时，需要区分：

两个循环可能执行相同次数，却产生不同结果。

# 10. Debugging Example<br>调试例子

The following loop is supposed to print:

下面的循环原本应该输出：

```text
5 3 1
```

Buggy code:

错误代码：

```java
for (int count = 1; count <= 5; count++)
{
    System.out.print(count + " ");
}
```

Actual output:

实际输出：

```text
1 2 3 4 5
```

The loop runs, but its logic does not match the required pattern. This is a **logic error**.

循环能够运行，但逻辑与目标不符。这属于**逻辑错误**。

Three parts must change:

三个部分都需要修改：

1. Start at `5`, not `1`.
   从 `5` 开始，而不是 `1`。

2. Continue while the value is at least `1`.
   当数值仍然大于或等于 `1` 时继续。

3. Decrease by `2`.
   每次减少 `2`。

### Fixed code

```java
for (int count = 5; count >= 1; count -= 2)
{
    System.out.print(count + " ");
}
```

```text
5 3 1
```

| Problem           | Effect                                                     | Fix                        |
| ----------------- | ---------------------------------------------------------- | -------------------------- |
| Starts at `1`     | Begins at the wrong endpoint.<br>从错误的端点开始。                 | Initialize `count` to `5`. |
| Uses `count <= 5` | Describes an increasing range.<br>描述的是递增范围。                | Use `count >= 1`.          |
| Uses `count++`    | Moves upward instead of downward by two.<br>向上增加，而不是每次减少二。 | Use `count -= 2`.          |

## Practice 4: Match the `while` Loop<br>练习四：匹配 `while` 循环

Which `while` loop is equivalent to this `for` loop?

下面哪个 `while` 循环与这段 `for` 循环等价？

```java
for (int i = 1; i <= 4; i++)
{
    System.out.print(i);
}
```

A.

```java
int i = 1;

while (i <= 4)
{
    i++;
    System.out.print(i);
}
```

B.

```java
int i = 1;

while (i <= 4)
{
    System.out.print(i);
    i++;
}
```

C.

```java
int i = 0;

while (i <= 4)
{
    System.out.print(i);
    i++;
}
```

**Answer: B**

The body uses the current value before the update, exactly like the `for` loop.

循环体先使用当前值，再执行更新，与原来的 `for` 循环一致。

## Practice 5: Find the Infinite Loop<br>练习五：找出无限循环

What is wrong with this loop?

下面的循环有什么问题？

```java
for (int i = 0; i < 5; i--)
{
    System.out.println(i);
}
```

**Answer:** The update moves `i` in the wrong direction.

**答案：**更新语句让 `i` 朝错误的方向变化。

The values become:

变量值会变成：

```text
0 -1 -2 -3 ...
```

The condition `i < 5` remains true, so the loop does not stop.

条件 `i < 5` 会始终为真，因此循环不会停止。

A possible fix is:

一种修复方式是：

```java
for (int i = 0; i < 5; i++)
{
    System.out.println(i);
}
```

# Quick Checklist<br>快速检查清单

Before answering a Topic 2.8 question, check:

做 2.8 的题目之前，检查：

* What is the loop variable’s initial value?
  循环变量的初始值是什么？

* What Boolean condition controls the loop?
  哪个布尔条件控制循环？

* Is the ending boundary included or excluded?
  是否包含结束边界？

* Does the condition use `<`, `<=`, `>`, or `>=`?
  条件使用的是 `<`、`<=`、`>` 还是 `>=`？

* How does the update change the loop variable?
  更新语句如何改变循环变量？

* Does the update run before or after the body?
  更新语句在循环体之前还是之后执行？

* What exact values does the loop variable take?
  循环变量具体会取哪些值？

* How many times does the body execute?
  循环体会执行多少次？

* Could two loops run the same number of times but use different values?
  两个循环是否可能执行相同次数，却使用不同数值？

* Does the update move toward making the condition false?
  更新是否会让条件逐渐变为假？

* Could the loop be infinite?
  循环是否可能成为无限循环？

* Is there an off-by-one error?
  是否存在差一错误？

* Are both semicolons present in the `for` header?
  `for` 循环头中的两个分号是否都存在？

* If converting to a `while` loop, is the update placed at the end of the body?
  改写成 `while` 循环时，更新是否位于循环体末尾？
