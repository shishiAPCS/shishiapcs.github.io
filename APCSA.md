## CB官方样题：
---

Consider the following code segment.
```
ArrayList<String> items = new ArrayList<String>(); 
items.add("A");
items.add("B");
items.add("C");
items.add(0, "D");
items.remove(3);
items.add(0, "E");
System.out.println(items);
```
What is printed as a result of executing the code segment?<br>
(A) [A,B,C,E]<br>
(B) [A,B,D,E]<br>
(C) [E,D,A,B]<br>
(D) [E,D,A,C]<br>
(E) [E,D,C,B]<br>
